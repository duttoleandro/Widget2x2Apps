# Widget 2x2 Apps

Widget Android 2x2 con cuatro aplicaciones configurables, fondo gris, bordes redondeados y transparencia ajustable.

## Compilación
Abre el proyecto en Android Studio y usa `Build > Build APK(s)`.
La APK estará en `app/build/outputs/apk/debug/app-debug.apk`.

También se incluye `.github/workflows/build-apk.yml` para compilación mediante GitHub Actions.
